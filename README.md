# 🤖 Minecraft AFK Bot с анти-AFK системой

Полнофункциональный AFK-бот для Minecraft сервера с встроенной системой против автоматического кика.

## 📋 Требования

- **Node.js** версия 14 или выше ([скачать](https://nodejs.org/))
- **npm** (идёт с Node.js)
- Доступ к Minecraft серверу

## 🚀 Установка и запуск

### Шаг 1: Установка зависимостей

Откой PowerShell или Terminal в папке проекта и выполни:

```powershell
npm install
```

Это установит все необходимые библиотеки.

### Шаг 2: Конфигурация

Отредактируй **bot.js** (строка 7-9):

```javascript
const SERVER_CONFIG = {
  host: 'SMP-FRUTLINE.aternos.me',  // IP сервера (в твоём случае - верно)
  port: 29506,                       // Порт (в твоём случае - верно)
  username: 'AFK_Bot',              // ИЗМЕНИ на своё имя игрока!
  version: null                      // Версия определяется автоматически!
};
```

✅ **Версия определяется автоматически** - не нужно указывать вручную!

### Шаг 3: Запуск бота

```powershell
npm start
```

Или:

```powershell
node bot.js
```

## 🛡️ Анти-AFK система

Бот выполняет несколько действий для предотвращения кика:

| Действие | Интервал | Цель |
|----------|----------|------|
| ⬇️ **Клик** | 15 сек | Активирует нежить, даёт сигнал об активности |
| 🔄 **Поворот камеры** | 20 сек | И меняет направление взгляда |
| 🚶 **Движение** | 40 сек | Передвижение в разных направлениях |
| ⬆️ **Прыжок** | 60 сек | Периодические прыжки |
| 📦 **Проверка инвентаря** | 2 мин | Открытие/закрытие инвентаря |

**Все интервалы можно изменить в ANTI_AFK_CONFIG (строка 16-22 в bot.js)**

## 💬 Команды в консоли

Во время работы бота можно вводить команды:

```
chat <сообщение>    - Отправить сообщение в чат сервера
coords              - Показать текущие координаты
health              - Показать здоровье бота
stop                - Остановить бота и выйти
help                - Показать справку
```

**Пример:**
```
> chat Привет, я AFK бот!
> coords
> stop
```

## 🔧 Расширенная конфигурация

Отредактируй **config.js** для более гибких настроек:

```javascript
// Изменение интервалов
antiAFK: {
  clickInterval: 15000,      // Клик каждые 15 сек
  rotateInterval: 20000,     // Поворот каждые 20 сек
  moveInterval: 40000,       // Движение каждые 40 сек
  jumpInterval: 60000        // Прыжок каждые 60 сек
}
```

**Совет:** Если сервер частый кикает:
- Уменьши интервалы (например, 10000 вместо 15000)
- Используй разные интервалы (чтобы было меньше подозрений)

## ⚙️ Версия Minecraft

**✅ Версия определяется автоматически!**

Бот сам определит версию сервера при подключении. Тебе не нужно ничего менять.

## 🐛 Решение проблем

### ❌ "Cannot find module 'mineflayer'"
Запусти установку зависимостей:
```powershell
npm install
```

### ❌ "Error: getaddrinfo ENOTFOUND SMP-FRUTLINE.aternos.me"
- Проверь IP и порт сервера
- Убедись, что сервер включен (если это Aternos, сервер должен работать)
- Проверь подключение к интернету

### ❌ "Failed to authenticate"
- Проверь username (имя должно быть зарегистрировано в Minecraft)
- Если сервер в режиме online-mode, убедись, что имя правильное

### ❌ Бот сразу отключается
- Проверь, что сервер включен (если это Aternos)
- Может быть, у ника нет доступа на сервер (чёрный список/белый список)
- Если проблема остаётся, можешь указать version вручную в SERVER_CONFIG
